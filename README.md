# Basic Documentation:
# git commit -m "message" → Saves your changes to the local repository with a short message.

# git push → Sends your local commits to the remote repository (like GitHub).

# git pull → Downloads and merges changes from the remote repository into your branch.

# git branch → Shows all local branches in your repository.

# git branch --all → Shows both local and remote branches.

# git fetch --all → Downloads updates from all remotes without merging them.

# git stash → Temporarily saves your uncommitted changes and cleans the working directory.

# git checkout -b newbranchname → Creates a new branch and switches to it.

#git checkout branchname → Switches to an existing branch.

# python -m venv venv → Creates a virtual environment named venv.

#python -m venv (name) → Creates a virtual environment named (name here).

#deactivate → Exits the currently active virtual environment.
